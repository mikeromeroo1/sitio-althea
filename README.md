# Althea Lease - Arrendamiento de Equipo Médico

Sitio web para Althea Lease, una empresa especializada en ofrecer soluciones de arrendamiento de equipo médico para profesionales de la salud en México. La plataforma facilita la cotización y el acceso a equipos médicos mediante un proceso claro y con diversos beneficios adicionales.

## Tecnologías Utilizadas

Este proyecto está construido con:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desarrollo Local

Para trabajar en este proyecto localmente:

1.  Clona el repositorio:
    ```sh
    git clone <URL_DEL_REPOSITORIO>
    ```
2.  Navega al directorio del proyecto:
    ```sh
    cd <NOMBRE_DEL_PROYECTO>
    ```
3.  Instala las dependencias:
    ```sh
    npm install
    ```
4.  Inicia el servidor de desarrollo:
    ```sh
    npm run dev
    ```

## Despliegue
